# @sequential/websocket-factory

WebSocket connection factory pattern for building subscription-based real-time endpoints.

## Installation

```bash
npm install @sequential/websocket-factory
```

## Usage

```javascript
import {{ main }} from '@sequential/websocket-factory';

const result = main(/* args */);
```

## API

### Main Export

Main function/class provided by this package.

## Examples

```javascript
// Basic usage
const value = main(input);

// With options
const result = main(input, {{ /* options */ }});
```
